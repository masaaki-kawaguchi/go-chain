# go-chain

## サーバー起動

```shell
go run main.go
```

## チェーンの内容
  
- ブロックナンバー  
- ナンス  
- ハッシュ
- 一つ前のハッシュ  
- トランザクション  
  
## トランザクションの内容  
  
- 送り手のアドレス  
- 受け手のアドレス  
- 額 
- メモ
- 署名
